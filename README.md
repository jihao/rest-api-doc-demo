## rest-api-doc-demo
An spring boot (java configured) web mvc app with Swagger2 + springfox

### Running
Run in STS as Spring Boot Application *or*

```
mvn spring-boot:run
```

### Open Swagger UI
http://localhost:8080/swagger-ui.html
